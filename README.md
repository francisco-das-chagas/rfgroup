# 🌐 RF Group 

> Uma Landing Page imersiva e interativa desenvolvida para unificar o ecossistema de empresas do RF Group.

![Project Preview](https://via.placeholder.com/1200x600?text=Preview+do+Site) 

## 📖 Sobre o Projeto

Este projeto é um Hub Corporativo desenvolvido para apresentar as 4 frentes de atuação do grupo: **Jurídico, Tecnologia, Imobiliário e Soluções Empresariais**.

O objetivo foi criar uma experiência de navegação fluida ("scrollytelling"), onde o design e a atmosfera do site mudam dinamicamente conforme o usuário explora cada seção, refletindo a identidade visual única de cada empresa subsidiária.

## 🚀 Tecnologias & Bibliotecas

* **HTML5 & CSS3:** Estrutura semântica e estilização moderna.
* **JavaScript (ES6+):** Lógica de interação e manipulação do DOM.
* **[GSAP](https://greensock.com/gsap/) (GreenSock):**
    * *ScrollTrigger:* Para animações sincronizadas com a rolagem e transições de cores de fundo.
* **[Lenis](https://github.com/studio-freight/lenis):** Para um "Smooth Scroll" (rolagem suave) de alta performance.
* **[tsParticles](https://particles.js.org/):** Para a criação do background geométrico interativo na seção inicial (triângulos conectados).
* **Lucide Icons (SVG):** Ícones vetoriais inline animados.

## ✨ Funcionalidades

* **🎨 Transição de Temas:** O fundo do site muda suavemente de cor (Branco → Vinho → Azul → Cinza → Preto) para combinar com a marca da seção visível.
* **❄️ Glassmorphism:** Cards com efeito de vidro fosco para destacar o conteúdo sobre fundos coloridos.
* **🔺 Partículas Interativas:** Malha geométrica animada na seção Hero.
* **🔢 Contadores Animados:** Números estatísticos que "contam" progressivamente ao entrarem na tela.
* **📱 Totalmente Responsivo:** Layout adaptado para Mobile, Tablet e Desktop, com ajustes específicos de posicionamento para garantir a legibilidade.
* **🖱️ Micro-interações:** Botões com efeitos de hover e ícones SVG desenhados via código.
